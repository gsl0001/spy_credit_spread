## 3. IBKR connectivity (paper port 7497 first)

## Rollback criteria (auto-stop if any fire in week 1)

| Condition                                       | Action                         |
|-------------------------------------------------|--------------------------------|
| Any unhandled exception in `/api/ibkr/execute`  | stop scanner, keep positions   |
| Daily P&L < −1% of equity                       | flatten all, switch to paper   |
| IBKR socket drops > 3× / day                    | stop scanner, investigate I7   |
| Backtest↔live signal divergence > 10% / day     | stop scanner, investigate C8   |
| Journal or DB write failure                     | stop everything, page operator |
